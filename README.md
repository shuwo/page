### Introduce

博客项目，使用hexo，页面部署到shuwo.github.io。

### Install

```
npm install hexo-cli -g
git clone https://github.com/shuwo/shuwo-page.git
cd shuwo-page/blog/
npm install
```

### Usage

文章写在`shuwo-page/blog/source/_posts/`中，其他操作（`shuwo-page/blog/`目录下执行命令）：
* 本地预览：hexo s
* 生成文件：hexo g
* 部署到shuwo.github.io：hexo d
